grep

grep -o -E 使用正则表达式并且只匹配单词
grep -w -E 使用正则表达式且打印出整行
grep --color 颜色匹配

在使用正则表达式时注意关键字，使用转义符号”"进行转义

getconf

getconf 查询系统属性变量

getconf LONG_BIT 打印出系统位数

tldr

tldr 取代man，命令查询工具

sudo apt install tldr - 下载安装
mkdir -p ~/.tldr/tldr && sudo git clone https://codechina.csdn.net/mirrors/tldr-pages/tldr.git ~/.tldr/tldr - 更换查询源
tldr --update 刷新缓存

sudo-一些小细节

通过sudo命令执行指令时，系统会重置当前用户的环境变量，重置为安全的环境变量。

sudo配置文件处于/etc/sudoers，可在此设置sudo
环境变量更改的原则。

使用sudo -E命令保留当前用户存在的环境变量。
