Linux文本处理

Linux三剑客：grep awk sed

sysctl

sysctl –> 用于Linux内核参数的调整，比如网络tcp连接的设置等。我们可以在cat /proc/sys/对应内核参数获取对应参数值

例如net.ipv4.tcp_keepalive_time属性，可以使用cat /proc/sys/net/ipv4/tcp_keepalive_time获取当前属性值

starce

strace 用于追踪程序执行的函数库调用过程，添加至执行的程序前

例如strace neofetch

Posix

概念：可移植操作系统接口（Portable Operating System Interface of UNIX）

最初用于标准化Unix，提高兼容性和应用程序的可移植性。

涵盖的范围包括：系统调用的C语言接口（C库），shell程序，线程等

Unix Domain Socket

又称IPC：inter-process communication socket

功能：实现同一主机上的进程间通信。

Unix Domain Socket是Posix标准中的组件，因此具有相当的兼容性和可移植性。

不同于网络传输中Socket的地址（Ip+Port），Unix Domain Socket中的Socket地址（相当于表示另一个进程的地址）为对应Socket类型文件在本地的地址。
