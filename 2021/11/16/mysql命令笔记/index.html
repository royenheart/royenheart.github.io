前言

安装

CentOS

教程来自：进入

步骤：

  1. 下载安装MySQL官方Yum Repository
  2. 安装MySQL服务器

Windows

  1. 进入官网下载对应文件。
  2. 安装包进行安装(Server 和 client)。
  3. 若需要在cmd或powershell等终端执行mysql连接，打开Windows的环境变量编辑，在系统Path变量中加入对应的mysql文件目录。
  4. mysql.exe默认安装情况下存放于C:\Program Files\MySQL\MySQL Server 8.0\bin目录。

指令

MySQL服务运行

指令 内容
systemctl start mysqld.service mysql服务启动。
systemctl status mysqld.service mysql服务状态查看。
service mysqld restart mysql服务重启

MySQL进入

指令 内容
grep “password” /var/log/mysqld.log 初次安装时查看设置的密码。
mysql -u root -p -u参数指定登录的用户名，注意此处的root并非系统中的root用户。
mysql -u root -p -p 指定密码，会让你输入密码进行登录，实际上可以将密码直接跟于参数后，但出于安全不推荐这样做。
mysql -h address -u user -p 远程连接对应主机的mysql库。
grant all privileges on . to ‘用户名‘@’主机名’ identified by ‘password’ with grant option; 在赋予权限时指定远程访问的密码。需要在开放对应端口，默认为3306。

MySQL用户设置

指令 内容
ALTER USER ‘用户名‘@’localhost’ IDENTIFIED BY ‘password’; 设置对应用户的密码。
create user 用户名 identified by ‘密码’; 创建新用户和对应密码。
grant 权限 on 数据库.数据表 to ‘用户名‘@’主机名’; 指定的数据库和表开放对应权限给指定用户。
grant all privileges on “.” to ‘用户名‘@’%’; 对所有数据库和表开放所有权限给任何主机登录的user用户。
grant all privileges on “.” to ‘用户名‘@’%’; 我们应该注意到从不同主机（MySQL支持远程登录）登录的同名账户实际上是不同的，因此我们可以指定以一个或多个主机地址的账户才能开放权限。
grant select,insert …… 开放多个权限，权限之间用’,’隔开
revoke 权限 on 数据库.数据表 from ‘用户名‘@’主机名’; (root用户)收回指定用户的相关权限，和grant相似，不过赋予权限改为收回权限。
flush privileges; 刷新保存权限的改动。

MySQL语言设置

步骤：

  1. 修改/etc/my.cnf
  2. 添加配置

    [client]
    default-character-set=utf8
    character-set-server=utf8
    collation-server=utf8_general_ci

  3. 保存文件并重启mysql

MySQL数据操作

指令 内容
show databases; 列出所有的数据库。
show tables; 列出当前使用的数据库存在的所有数据表。
select * from < 数据表名 >; 列出指定数据表中的内容。
create database < name >; 创建指定名字的数据库。
CREATE TABLE IF NOT EXISTS table name( id INT UNSIGNED AUTO_INCREMENT, data1 VARCHAR(20) NOT NULL, PRIMARY KEY (id))ENGINE=InnoDB DEFAULT CHARSET=utf8; 创建数据表。先设置各个记录的字段即数据项，再设置主键，最后设置驱动引擎和表所使用的编码集。VARCHAR(20)表示20字符长度的字符串，NOT NULL表示字符串不得为空，INT表示整形数据，UNSIGNED表示无符号类型，AUTO_INCREMENT表示未指定值时按照已有数据递增，一般用于主键。
INSERT INTO < table name >(data1,…,datan)VALUES(value1,……,valuen); 往指定数据表中插入数据，注意数据的格式以及主键。
drop database < name >; 删除对应的数据库。
drop table < name >; 删除对应的数据表。
use < databse name >; 切换当前使用的数据库。
update 表名 set 字段名=新值 条件; 当符合给定条件时修改数据表中各个记录对应字段的值。条件写为where statement
alter table 表名 add 新字段; 在数据表中添加新的字段
desc 表名; 查看数据表的结构

MySQL数据迁移

指令 内容
mysqldump -u user -p dbname > dbname.sql 将对应数据库数据导出
mysql -h adress -u user -p < dbname.sql 将到处的数据库数据迁移至对应的数据库（可以跨服务器）。
